## Client

### Install

```bash
cd client
npm install
```

### Run

```bash
npm start
```

### Run Development

```bash
npm run dev
```

---

## Server

### Install

```bash
cd server
npm install
npx playwright install
```

### Run

```bash
npm start
```

### Run Development

```bash
npm run dev
```

### Test

```bash
npm test
```

#### Compile and Run Node Addons

```bash
npm run build-addons
```
